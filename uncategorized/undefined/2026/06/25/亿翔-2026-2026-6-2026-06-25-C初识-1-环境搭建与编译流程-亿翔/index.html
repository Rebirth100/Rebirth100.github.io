三、第一个完整可运行HelloWorld程序

代码逐行说明

  1. #include <stdio.h>：预编译指令，引入标准输入输出头文件，预编译阶段做文本替换
  2. #：预编译指令标识
  3. include：引入头文件关键字
  4. <>：系统头文件边界符
  5. int：函数返回值整型类型
  6. main：程序唯一入口函数
  7. ()：括号内void代表无参数
  8. {}：包裹函数体，代码缩进统一4个空格
  9. printf("hello,world!");：格式化输出函数；双引号包裹字符串；;代表语句结束
  10. return 0;：主函数正常结束返回0，标准规范不可省略
  11. //：单行注释

补充常用2次幂速记

  • 2⁸ = 256
  • 2¹⁰ = 1024
  • 2¹² = 4096
  • 2¹⁶ = 65536

补充取值范围总结

  1. signed char：-128 ~ 127
  2. unsigned char：0 ~ 255

十、基础整型short / int / long / long long

1. short（2字节，默认有符号）

  • 有符号范围：-2¹⁵ ~ 2¹⁵ - 1 → -32768 ~ 32767
  • 无符号unsigned short：0 ~ 2¹⁶ - 1 → 0 ~ 65535
  • 对应宏：SHRT_MIN / SHRT_MAX

2. int（4字节，默认有符号）

  • 有符号范围：-2³¹ ~ 2³¹ - 1 → -2147483648 ~ 2147483647
  • 无符号unsigned int：0 ~ 2³² - 1

3. long

平台相关：Windows 4字节，Linux/macOS 8字节

4. long long（8字节）

长整型，存储超大整数

补充：sizeof与%zu

sizeof(数据类型/变量)：计算内存占用字节数

  • 数据类型必须加括号：sizeof(int)
  • 变量可省略括号：sizeof(num)
  • 返回值类型：size_t（无符号整数），打印必须用%zu占位符

十一、浮点类型float / double

1. float（4字节）

  • 有效精度：6位十进制有效数字
  • 精度宏：FLT_DIG

2. double（8字节）

  • 有效精度：15位十进制有效数字
  • 精度宏：DBL_DIG

补充：左值截断规则

赋值时右值范围超过左值类型存储上限，自动发生二进制截取，数值错乱。

十二、字符串与编码规则

  1. 字符串使用双引号""包裹，末尾自动隐藏结束符'\0'
  2. 字符存储大小
    • 英文字母/数字：1字节
    • 汉字：GBK编码2字节；UTF-8编码3字节

\n 回车换行
\a 蜂鸣响铃
\t 水平制表符（Tab）
\r 回车，光标回到本行开头不换行
\0123 八进制转义字符
\x0a 十六进制转义字符

十四、printf格式化输出全解

通用格式：%[标志][宽度].[精度][长度][类型]

1. 基础占位符

占位符 适用类型 作用
%d int/char/short 十进制有符号整数
%u unsigned系列 十进制无符号整数
%c char 单个字符
%f float/double 浮点数
%s char数组字符串 打印字符串
%zu size_t 打印sizeof返回值
%p 任意变量地址 打印内存地址

2. 格式修饰符

  1. 标志位
    • -：左对齐，默认右对齐
    • 0：空位补0，默认补空格
  2. 宽度：控制输出总字符长度
  3. 精度
    • 浮点%f：控制小数点后位数
    • 字符串%s：截取前N个字符

3. puts简化输出

puts("字符串")：仅支持打印字符串，自动追加换行\n
