补充常用2次幂速记

  • 2⁸ = 256
  • 2¹⁰ = 1024
  • 2¹² = 4096
  • 2¹⁶ = 65536

补充取值范围总结

  1. signed char：-128 ~ 127
  2. unsigned char：0 ~ 255

十、基础整型short / int / long / long long

1. short（2字节，默认有符号）

  • 有符号范围：-2¹⁵ ~ 2¹⁵ - 1 → -32768 ~ 32767
  • 无符号unsigned short：0 ~ 2¹⁶ - 1 → 0 ~ 65535
  • 对应宏：SHRT_MIN / SHRT_MAX

2. int（4字节，默认有符号）

  • 有符号范围：-2³¹ ~ 2³¹ - 1 → -2147483648 ~ 2147483647
  • 无符号unsigned int：0 ~ 2³² - 1

3. long

平台相关：Windows 4字节，Linux/macOS 8字节

4. long long（8字节）

长整型，存储超大整数

补充：sizeof与%zu

sizeof(数据类型/变量)：计算内存占用字节数

  • 数据类型必须加括号：sizeof(int)
  • 变量可省略括号：sizeof(num)
  • 返回值类型：size_t（无符号整数），打印必须用%zu占位符

十一、浮点类型float / double

1. float（4字节）

  • 有效精度：6位十进制有效数字
  • 精度宏：FLT_DIG

2. double（8字节）

  • 有效精度：15位十进制有效数字
  • 精度宏：DBL_DIG

补充：左值截断规则

赋值时右值范围超过左值类型存储上限，自动发生二进制截取，数值错乱。

十二、字符串与编码规则

  1. 字符串使用双引号""包裹，末尾自动隐藏结束符'\0'
  2. 字符存储大小
    • 英文字母/数字：1字节
    • 汉字：GBK编码2字节；UTF-8编码3字节

\r 回车，光标回到本行开头不换行

十四、printf格式化输出全解

通用格式：%[标志][宽度].[精度][长度][类型]

1. 基础占位符

占位符 适用类型 作用
%d int/char/short 十进制有符号整数
%u unsigned系列 十进制无符号整数
%c char 单个字符
%f float/double 浮点数
%s char数组字符串 打印字符串
%zu size_t 打印sizeof返回值
%p 任意变量地址 打印内存地址

2. 格式修饰符

  1. 标志位
    • -：左对齐，默认右对齐
    • 0：空位补0，默认补空格
  2. 宽度：控制输出总字符长度
  3. 精度
    • 浮点%f：控制小数点后位数
    • 字符串%s：截取前N个字符

3. puts简化输出

puts("字符串")：仅支持打印字符串，自动追加换行\n

C语言案例分析：学生信息输入与输出

一、案例功能概述

本程序实现学生基本信息与两门科目成绩的输入，并按照整齐的表格格式输出学生信息，包含姓名、学号、科目名称及对应成绩，是C语言基础输入输出、数组、数据类型的综合应用案例。

二、核心思路设计

  1. 数据类型选择
    • 姓名、科目名称：使用字符数组存储字符串
    • 学号：使用无符号整型存储纯数字学号
    • 成绩：使用浮点型存储带小数的分数
  2. 流程设计
    输入姓名 → 输入学号 → 输入科目1及成绩 → 输入科目2及成绩 → 格式化表格输出所有信息

三、完整可运行代码

四、代码逐段解析

1. 头文件

  • 作用：引入标准输入输出库，提供printf、scanf等输入输出函数。

2. 主函数

  • 程序入口，所有逻辑在main中执行，return 0表示程序正常结束。

3. 变量定义部分

变量名 类型 作用说明
name[20] 字符数组 存储学生姓名，长度20，存放字符串
id unsigned int 存储学号，无符号整型，仅存正整数
sub1[10] 字符数组 存储第一门科目名称
sub2[10] 字符数组 存储第二门科目名称
chinese_score float 存储第一门科目成绩（小数）
math_score float 存储第二门科目成绩（小数）

4. 信息输入部分

  • printf：提示用户输入内容
  • scanf("%s", name)：读取字符串存入name，数组名本身是地址，无需加&
  • %u：匹配unsigned int类型，比%d更规范
  • 普通变量需加&取地址
  • %s动态输出科目名称，提升交互友好性
  • %f用于读取浮点型成绩

5. 格式化输出部分

格式符 含义
%-8s 左对齐，占8个字符宽度，字符串输出
%-12u 左对齐，占12位，无符号整型输出
%-8.2f 左对齐，占8位，保留2位小数，浮点数输出
\n 换行，保证表格分行显示
  • 左对齐-：让表格列对齐更整齐
  • .2f：成绩保留两位小数，符合日常分数显示习惯

六、注意事项

  1. scanf("%s") 无法读取带空格的字符串，姓名/科目含空格需改用fgets
  2. unsigned int 建议使用%u匹配，避免警告
  3. 数组长度需预留足够空间，防止字符串溢出
  4. 输出格式宽度统一，才能实现整齐的表格效果
