三、指针基础

1. 指针本质

指针 = 变量存储内存地址。

  • int *ptr：int 代表指针指向空间存储int类型；* 标识ptr是指针变量。
  • & 取地址符，* 解引用符，二者互为逆运算。

4. 指针大小

指针存储空间大小由操作系统位数决定：

  • 64位系统：所有指针占8字节
  • 32位系统：所有指针占4字节

5. 野指针 & 空指针

  1. 野指针：未初始化的指针，地址随机，禁止解引用，会产生未定义行为(UB)
  2. 空指针：赋值为NULL，代表无有效指向，同样禁止解引用

底层指针运算拆解

  1. arr：首行指针
  2. arr + i：偏移i行，指向第i行
  3. *(arr + i)：解引用，转为该行首元素指针
  4. *(arr + i) + j：偏移j列，指向第j个元素
  5. *(*(arr + i) + j)：取出元素值

七、数组指针 vs 指针数组

1. 数组指针 int (*p)[3]

  • 本质：指针
  • 作用：指向一整行数组，适配连续存储的二维数组
  • 内存特点：二维数组行、列内存连续

2. 指针数组 int *p[3]

  • 本质：数组，数组内每个元素是int类型指针
  • 内存特点：每行地址不一定连续
