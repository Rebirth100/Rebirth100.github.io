Python学习第一天笔记

一、开发环境与HelloWorld

  1. 核心工具
    • Python解释器：负责执行代码
    • PyCharm：主流Python开发IDE
  2. 编码规范
    • 文件开头声明编码：# coding:utf-8，确保中文正常显示
  3. 第一个程序

二、注释语法

1. 作用

  • 临时屏蔽代码
  • 为代码添加说明，提升可读性 ### 2. 分类
  • 单行注释：以 # 开头，快捷键 Ctrl+/
  • 多行注释：使用三单引号 ''' ''' 或三双引号 """ """ ### 3. 版本标记块

三、输入输出

1. 输出 print()

  • 默认换行，可通过 end 参数自定义结尾
  • 多参数自动用空格分隔
  • 字符串拼接仅支持同类型，需用 + 连接

2. 输入 input()

  • 阻塞程序，等待用户回车后继续执行
  • 返回值固定为 str 类型，数字运算需手动转换

四、变量与数据类型

1. 变量命名规则

  • 由字母、数字、下划线组成，不能以数字开头
  • 推荐使用英文单词，布尔变量建议以 is 开头（如 is_student）

2. 基础数据类型

类型 说明 示例
int 整型（支持二进制/八进制/十六进制） a = 10, b = 0b101
float 浮点型 c = 3.14, d = 2e3
complex 复数 e = 3+4j, e.real
bool 布尔型 f = True, bool(0) == False
str 字符串 g = "Python", h = '''多行文本'''
None 空值 i = None
  • 字符串特性：
    • 转义字符：\t（制表符）、\n（换行）
    • 原始字符串：r"" 不解析转义字符
    • 引号嵌套：print("我说：\"你好\"")

五、运算符

1. 算术与赋值运算符

运算符 作用 示例
+ - * / 加减乘除 5 + 3
// 整除 7 // 2 = 3
% 取余 7 % 2 = 1
** 幂运算 2 ** 3 = 8
+= -= *= /= 复合赋值 a += 1 等价于 a = a + 1
  • 浮点数精度问题：0.1 + 0.2 ≠ 0.3，可使用 decimal 模块精确计算

2. 比较运算符

> < >= <= == !=，结果为布尔值 True/False

3. 逻辑运算符

  • and：全真为真
  • or：一真即真
  • not：取反

4. 成员运算符 in / not in

判断元素是否存在于可迭代对象（字符串、列表等）

5. 身份运算符 is / is not

判断两个变量是否指向同一块内存地址 - ==：比较值是否相等 - is：比较内存地址是否相同

六、字符串格式化

1. 四种方式

  • 拼接法："姓名：" + name
  • %格式化："年龄：%d，余额：%.2f" % (age, balance)
  • format格式化："{} {}".format(name, age) 或 "{n} {a}".format(n=name, a=age)
  • f-string（推荐）：f"姓名：{name}，余额：{balance:.2f}"

七、字节串 bytes

  • 作用：用于网络传输、文件读写等二进制场景

  • 编码长度：英文/数字占1字节，中文（UTF-8）占3字节
