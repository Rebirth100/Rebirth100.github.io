2. 替换 replace

  • replace(旧内容, 新内容)：全局全部替换
  • replace(旧内容, 新内容, 替换次数)：指定替换前N个

3. 首尾判断 startswith / endswith

返回布尔值 True / False

  • startswith("内容")：判断字符串是否以指定内容开头
  • endswith("内容")：判断字符串是否以指定内容结尾

4. 大小写转换与校验

方法 功能说明
upper() 全部字母转大写
lower() 全部字母转小写
title() 每个单词首字母大写，其余小写
capitalize() 仅字符串第一个字符大写，其余小写
istitle() 判断是否每个单词首字母大写
isupper() 判断字符串是否纯大写字母
islower() 判断字符串是否纯小写字母

5. 去除空格 strip / lstrip / rstrip

  • strip()：清除字符串左右两端空白
  • lstrip()：清除左侧空白
  • rstrip()：清除右侧空白

6. 分割 split 与拼接 join

split(分隔符, 分割次数)

将字符串切割为列表，分割次数可选，分割后列表数量=次数+1

join(可迭代对象)

使用指定字符将列表、元组等内容拼接成完整字符串

7. 数字判断 isdigit()

  • isdigit()：判断字符串是否全部由纯数字组成
  • 小数校验思路：删除最多1个小数点后再判断是否纯数字

二、if-else条件分支语句（03ifelse.py）

1. 单if分支

仅满足条件时执行代码，不满足无操作

2. if-else 双分支

二选一逻辑，条件成立执行if，不成立执行else

3. if-elif-else 多分支

从上到下依次匹配条件，匹配到第一个成立条件后直接结束判断

4. 嵌套if语句

if内部再嵌套if，用于多层条件校验

三、三目（三元）运算符（04三目运算符.py）

简化双分支if-else，一行代码完成判断

基础语法

结果A if 条件 else 结果B
条件成立返回A，不成立返回B

四、while循环基础（05循环.py）

循环作用：重复执行一段代码

  • while循环：灵活通用，适合不确定循环次数场景
  • for循环：专门遍历可迭代数据

while循环四要素

  1. 初始化循环变量
  2. 循环条件判断
  3. 循环体执行代码
  4. 更新循环变量（不更新会产生死循环）

死循环 while True

条件永久成立，常用于程序持续交互、游戏主程序

五、break / continue / 循环else（06break_continue_else.py）

  1. break：直接终止并跳出整个循环
  2. continue：跳过当前这一轮循环，直接进入下一轮
  3. 循环else：仅当循环正常走完（未被break打断）才执行；break跳出则不执行else

九、课后综合作业（HOMEWORK.txt）

作业1：无正则验证邮箱

校验规则：

  1. 必须同时包含 @ 和 .
  2. @ 必须出现在 . 的前面
  3. 不能以 @ 开头，不能以 . 结尾
  4. @ 和 . 不能相邻

作业2：彩票金额计算

规则：

  • 010203040506#07 → 2元
  • 010203040506#07#3 → 2*3=6元（末尾数字为倍数）
  • 010203040506#070811 → 6元（多组号码）
  • 010203040506#070811#10 → 6*10=60元

作业3：BMI指数计算

公式：BMI = 体重 / (身高 ** 2)
分级标准：

  • <18.5：过轻
  • 18.5 ~ 25：正常
  • 25 ~ 28：过重
  • 28 ~ 32：肥胖

  • 32：严重肥胖

作业4：输出1~100同时被3和7整除的数字

判断条件：i % 3 == 0 and i % 7 == 0

作业5：棋盘芝麻重量计算

棋盘64格：
第1格1粒，第2格2粒，第3格4粒，每一格数量翻倍；单粒芝麻0.00001kg
计算全部芝麻总重量
